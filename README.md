# Qt-CustomControls
-个人用Qt封装了一些控件。
--全部是Qt painter自绘，无任何第三方库。
--设置的接口完备，有一些可能需要微调。
--基本上一个控件一个头文件加一个实现文件。
--分了一些类，现在有进度条，滑动条，仪表盘，统计图，动画，按钮，还有一些还不错的软件页面。
